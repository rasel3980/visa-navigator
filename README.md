# Visa Navigator
Welcome to the **Visa Navigator**, a comprehensive platform designed to help users easily access information about various visa types, application processes, and requirements for different countries. The website also allows users to apply for visas directly through the platform. 
**Live Site URL:** [https://visa-navigator-7c020.web.app/] 

## Features

- **User Authentication**: Register and log in securely using email/password or Google login.
- **View Visa Information**: Browse through the latest visa information from various countries including visa types, processing times, fees, and required documents.
- **Add New Visa**: Users can add new visa information (Country, Visa Type, Documents, etc.) after logging in.
- **Apply for a Visa**: Logged-in users can apply for visas by filling out an application form specific to each visa.
- **User Dashboard**: Users can view their added visas and applications, with the ability to update or delete their own visa data.

## Technologies Used

- **Frontend**: React.js, React Router, React Context for state management, Lottie Animations, Bootstrap for styling.
- **Backend**: Node.js, Express.js, MongoDB for storing visa and user data.
- **Authentication**: Firebase Authentication for login, registration, and social login (Google).
- **Deployment**: Frontend hosted on Firebase, Backend API hosted on Vercel.
